# complex-schrodinger-wavefunction-predictor

# Objective
Develop a Complex-Valued Neural Network (CVNN) that predicts the eigenfunctions and eigenvalues of the 1D Time-Independent Schrödinger Equation for arbitrary potential energy functions.
